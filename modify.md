**홈 화면 UI 개선**
 - 홈 화면에서 하단 설정 버튼 이동이 있는데 이 버튼은 불필요해 보여. 앱바 우측에 설정 버튼이 있기 때문에 중복으로 존재하는 것 같아. 홈 화면에서 하단 설정 버튼을 제거하는 것을 제안해.
 - 중간에 있는 pen marker 이런 칩셋 위젯들이 정렬이 안되어 있어. 오히려 불필요한 공간이 많이 생겨서 UI가 깔끔하지 않아 보여. 이 부분을 개선해서 칩셋 위젯들이 정렬되도록 하는 것을 제안해. 그리고 더 작게 만들어서 공간을 더 효율적으로 사용할 수 있도록 하는 것도 고려해보면 좋을 것 같아.
 - 그리고 Saved Drawings 부분도 굳이 이렇게 큰 영역을 차지할 필요가 없어 보여. 이 부분도 더 작게 만들어서 공간을 더 효율적으로 사용할 수 있도록 하는 것을 제안해. 그리고 Saved Drawings 부분이 홈 화면에서 너무 눈에 띄는 것 같아. 이 부분을 좀 더 덜 눈에 띄게 만들어서 홈 화면이 더 깔끔하게 보이도록 하는 것도 고려해보면 좋을 것 같아.

**히스토리 화면 UI 개선**
 - 홈 화면 및 앱 테마에 맞게 더 세련되게 디자인해줘. 현재 히스토리 화면은 홈 화면과 앱 테마에 맞지 않는 디자인으로 되어 있어서 전체적인 앱의 일관성이 떨어지는 것 같아. 히스토리 화면도 홈 화면과 앱 테마에 맞게 더 세련되게 디자인하는 것을 제안해.
