## 개선안

**프리미엄 화면의 후원 문구 수정**
 - 후원 선택 화면에서 작은 후원/ 중간 후원/ 큰 후원으로 되어 있는데 이 부분을 Coffee/Lunch/Dinner로 변경하는 것을 제안드립니다. 당연히 각 로케일에 맞게 변경되어야 할 것입니다. (예: 한국어에서는 커피/점심/저녁으로 변경) 

**앱바 타이틀 스타일 통일**
 - 홈 화면의 앱바 타이틀의 Text 스타일을 확인해서 다른 화면의 앱바 타이틀과 동일한 스타일로 변경하는 것을 제안드립니다. 홈 화면의 앱바 타이틀이 다른 화면과 스타일이 달라서 일관성이 떨어지는 것 같습니다.

**홈 화면에서 불필요한 단계 줄이기**
 - 홈 화면에서 온보딩 기능 같은 화면에서 그리기 시작 버튼을 눌러야만 홈 화면으로 이동하는데, 이런 단계로 그릴 수 있는 화면으로 가는건 이상해.. 바로 홈 화면으로 이동하는 것을 제안드립니다. 온보딩 기능은 앱을 처음 실행할 때만 보여주고, 이후에는 바로 홈 화면으로 이동하도록 하는 것이 좋을 것 같습니다.

**캔버스 색상 선택에서 커스텀 색상 기능 추가**
 -pub.dev에서 컬러 피커를 제공하는 패키지를 찾아서, 캔버스 색상 선택에서 기본 색상 외에 커스텀 색상을 선택할 수 있는 기능을 추가하는 것을 제안드립니다. 사용자가 원하는 색상을 직접 선택할 수 있도록 하면 더 다양한 표현이 가능해질 것입니다. 아니면 하단에서 커스텀 색상 선택 할 수 있는 부분을 참고해서 색상을 사용자가 선택 할 수 있게 해주세요.

**하단 색상 선택에서 색상 선택은 가능한데 선택되어져 있다는 표시가 없음**
 - 하단 색상 선택에서 색상을 선택하면 해당 색상이 선택되어져 있다는 표시가 없어서, 사용자가 현재 어떤 색상이 선택되어 있는지 알기 어렵습니다. 선택된 색상에 테두리를 추가하거나, 선택된 색상에 체크 표시를 추가하는 등의 방법으로 선택된 색상을 시각적으로 표시하는 것을 제안드립니다.

**갤러리에 저장 UI 개선**
- 해상도 선택은 불필요하고 괜히 복잡도만 올라가니 삭제해주고 포맷 선택만 필요할 것 같습니다. 그리고 포맷 선택을 바텀시트의 넓이를 최대한 활용해서 가로로 나열하는 형태로 변경하는 것을 제안드립니다. 지금은 왼쪽에 몰려 있고 작게 보여서 선택하기 어려운 것 같습니다. 

**캔버스 줌인 / 줌 아웃 기능**
 - 캔버스에 손가락 두 개로 핀치 인/아웃 제스처를 추가해서, 사용자가 캔버스를 확대하거나 축소할 수 있도록 하는 것을 제안드립니다. 이렇게 하면 세밀한 부분을 그릴 때 편리할 것입니다.
