# Google Play 스크린샷 가이드

## 스크린샷 개요
- 짧은 제목 + 바로 이해되는 기능 문구로 구성
- 1장: 홈 화면과 빠른 시작
- 2장: 풀스크린 드로잉 캔버스
- 3장: 브러시와 색상 팔레트
- 4장: 저장과 작품 갤러리
- 5장: 작품 갤러리와 설정
- 6장: 후원형 Premium

---

## 스크린샷 구성 (6장)

### 1장: 홈 화면과 빠른 시작
- **제목(KO)**: **바로 그리기**
- **부제목(KO)**: **깨끗한 드로잉 시작**
- **Title(EN)**: **Start Drawing**
- **Subtitle(EN)**: **Clean canvas**

**캡처 화면**: 홈 화면에서 앱 이름, 핵심 기능 카드, `Start Drawing`과 `My Artworks` 진입이 보이는 상태
- **포함 요소**:
  - 홈 주요 CTA(`Start Drawing`)와 작품 보관함 진입(`My Artworks`)
  - 브러시 / 색상 / 저장 / 공유 기능 요약
  - 무료 버전이면 하단 배너 광고 영역

---

### 2장: 풀스크린 드로잉 캔버스
- **제목(KO)**: **넓은 캔버스**
- **부제목(KO)**: **방해 없이 그리기**
- **Title(EN)**: **Full Canvas**
- **Subtitle(EN)**: **Draw with focus**

**캡처 화면**: 실제 선이 그려진 드로잉 화면
- **포함 요소**:
  - 넓은 흰색 캔버스
  - 저장 / 공유 / 실행 취소 / 다시 실행 버튼
  - 브러시 툴바
  - 화면에 실제 그림이 그려져 있는 상태

---

### 3장: 브러시와 색상 팔레트
- **제목(KO)**: **10종 브러시**
- **부제목(KO)**: **펜·연필·마커·수채화·에어브러시 외**
- **Title(EN)**: **10 Brushes**
- **Subtitle(EN)**: **Pen, pencil, marker, and more**

**캡처 화면**: 브러시 선택 영역과 색상 팔레트가 함께 보이는 화면
- **포함 요소**:
  - 10종 브러시 셀렉터(펜, 연필, 마커, 붓, 형광펜, 만년필, 크레파스, 수채화, 에어브러시, 지우개)
  - 16색 팔레트 + 커스텀 컬러
  - 브러시 크기 슬라이더
  - 보상형 광고 또는 Premium으로 특수 브러시를 해금할 수 있는 맥락

---

### 4장: 저장과 작품 갤러리
- **제목(KO)**: **저장하고 이어 그리기**
- **부제목(KO)**: **갤러리에서 다시 열기**
- **Title(EN)**: **Save and Reuse**
- **Subtitle(EN)**: **Gallery ready**

**캡처 화면**: 앱 내 작품 갤러리 화면에서 저장된 그림 그리드가 보이는 상태
- **포함 요소**:
  - 저장된 작품 썸네일 2열 그리드와 작품 수 표시
  - 작품 열기(탭) / 삭제(길게 누름) 동작
  - 기기 갤러리 저장 시 PNG·JPEG 포맷과 해상도 선택 맥락

---

### 5장: 작품 갤러리와 설정
- **제목(KO)**: **내 작품 보관함**
- **부제목(KO)**: **저장한 그림과 설정**
- **Title(EN)**: **My Artworks**
- **Subtitle(EN)**: **Saved drawings and settings**

**캡처 화면**: 작품 갤러리 그리드 또는 설정 화면
- **포함 요소**:
  - 저장된 작품 썸네일 그리드
  - 언어, 진동, 브러시 가이드, 지우기 전 확인, 흔들어 지우기, 저장 해상도/포맷, 로컬 데이터 관리 설정

---

### 6장: 후원형 Premium
- **제목(KO)**: **광고 없이 그리기**
- **부제목(KO)**: **한 번 후원으로 깔끔하게**
- **Title(EN)**: **Ad-Free Drawing**
- **Subtitle(EN)**: **One-time support**

**캡처 화면**: Premium 화면
- **포함 요소**:
  - Small / Medium / Large 3개 후원 옵션
  - Medium 기본 선택 또는 인기 배지
  - 광고 제거 / Premium 브러시 접근 혜택 영역
  - 구매 복원 버튼
  - 하단 고정 구매 CTA

## 캡처 팁

### 일반 가이드라인
- **해상도**: 최소 `1080 x 1920`
- **형식**: PNG 권장
- **비율**: 9:16 우선
- **기기**: 실제 Android 기기 캡처 권장
- **상태**: 빈 화면보다 실제 그림이 있는 화면을 우선 사용
- **테마**: 밝은 테마 우선, 색상 팔레트가 잘 보이게 촬영

### 강조할 요소
- 앱을 열고 바로 그릴 수 있는 흐름
- 10종 브러시와 16색 팔레트 + 커스텀 컬러
- Undo / Redo로 실험하기 쉬운 구조
- PNG·JPEG 기기 갤러리 저장, 앱 내 작품 보관함, 공유
- 11개 언어와 라이트/다크 테마
- 후원형 Premium의 3개 옵션과 광고 제거

### 스크린샷 배치 순서
1. 홈 화면과 빠른 시작
2. 풀스크린 드로잉 캔버스
3. 브러시와 색상 팔레트
4. 저장과 작품 갤러리
5. 작품 갤러리와 설정
6. 후원형 Premium

## 추가 권장사항

### 타겟 사용자
- 간단히 그림을 그리고 저장하고 싶은 사용자
- 아이디어를 그림으로 빠르게 남기고 싶은 사용자
- 손가락으로 낙서하거나 색칠하고 싶은 사용자
- 메신저나 SNS로 그림을 바로 공유하고 싶은 사용자
- 광고 없는 깔끔한 캔버스를 원하는 사용자

### 핵심 가치 제안
- 앱 실행 후 바로 그리기
- 10종 브러시 제공 (펜·연필·마커·붓·형광펜·만년필·크레파스·수채화·에어브러시·지우개)
- 16색 팔레트 + 커스텀 컬러와 브러시 크기 조절
- 기기 갤러리 저장(PNG·JPEG), 앱 내 작품 보관함, 다시 열어 편집, 공유까지 한 흐름
- 1회 후원으로 광고 제거와 Premium 브러시 접근

### 차별화 요소
- 드로잉에 필요한 기능만 간결하게 제공
- 저장한 작품을 갤러리에서 다시 열어 이어 편집 가능 (참조 사진도 함께 보존)
- 보상형 광고 또는 Premium으로 특수 브러시 접근 가능
- 11개 언어 지원
- 후원형 3단계 구매 구조로 혜택은 동일하게 유지
